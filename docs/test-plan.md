# Test Plan for E-Commerce Website Automation

## Project Overview
**Application**: Automation Practice E-Commerce
**URL**: http://automationpractice.multiformis.com
**Scope**: Registration, Login, Product Search
**Testing Tool**: Playwright with JavaScript

## Objectives
1. Verify user registration functionality
2. Validate login processes
3. Test product search features
4. Identify and report defects

## Testing Approach
- Automated functional testing
- Cross-browser testing (Chrome, Firefox)
- Data-driven testing for multiple scenarios
- Screenshot and video evidence collection

## Schedule
- Setup: Day 1
- Test Development: Days 2-3
- Execution & Reporting: Day 4
- Final Report: Day 5

## Deliverables
1. Automated test scripts
2. Test execution reports
3. Bug reports
4. Test summary report
